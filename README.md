# Web-crawler
Project for crawling an internal links written with JavaScript in the node.js environment.

## Tests
Unit tests are available:
```
npm test
```
## Examples
### First example
Input:
```
npm start http://sebastianbrzustowicz.pythonanywhere.com
```
Output: 
```

=========
```
